## 🧠 Core Concept

This lecture introduces the **foundation of all programming: data handling**.

In Python, variables are not just containers — they are **the building blocks of logic, memory, and computation**.

---

## ⚡ What You Will Learn

- How to create and use variables  
- Writing clean and meaningful variable names  
- Understanding core data types:
  - `int` → Whole numbers  
  - `float` → Decimal numbers  
  - `str` → Text values  
  - `bool` → Logical values (True/False)  
- Using `type()` to inspect data
