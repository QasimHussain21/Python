## 🧠 Core Concept

A **Dictionary** is a Python data structure used to store data in **key-value pairs**.

This lecture is not just about storing information,  
it’s about understanding **fast data access, organized records, and real-world data mapping**.

---

## ⚡ What You Will Learn

- What is a Dictionary in Python  
- Why dictionaries are used  
- Creating dictionaries  
- Accessing values using keys  
- Updating items  
- Adding and removing data  
- Dictionary methods  
- Looping through dictionaries
