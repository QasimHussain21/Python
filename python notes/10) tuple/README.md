## 🧠 Core Concept

A **Tuple** is a Python data structure used to store multiple values in a single variable.

This lecture is not just about storing data,  
it’s about understanding **immutable collections, faster data storage, and organized grouping of values**.

- What is a Tuple in Python

- Creating tuples  
- Accessing tuple values  
- Tuple slicing  
- Tuple methods  
- Packing and unpacking tuples
