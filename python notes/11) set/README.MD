## 🧠 Core Concept

A **Set** is a Python data structure used to store multiple unique values in one variable.

This lecture is not just about storing items,  
it’s about understanding **unordered collections, duplicate removal, fast membership checking, and set operations**.

---

## ⚡ What You Will Learn

- What is a Set in Python  
- Creating sets  
- Unique values and duplicates removal  
- Unordered nature of sets  
- Adding and removing items  
- Looping through sets  
- Set methods  
- Union, intersection, and difference  
- Finding max and min values
