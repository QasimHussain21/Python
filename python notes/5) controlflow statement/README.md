## 🧠 Core Concept

Control flow statements allow your program to **make decisions and control execution flow**.

They turn simple code into **dynamic, logic-driven systems**.

---

## ⚡ What You Will Learn

- Conditional statements  
- Decision-making logic  
- Writing dynamic programs  

---

## 🔀 Types of Control Flow

### 🔹 if Statement

```python
age = 18

if age >= 18:
    print("You can vote")
🔹 if-else Statement
age = 16

if age >= 18:
    print("Adult")
else:
    print("Minor")
🔹 if-elif-else Ladder
marks = 85

if marks >= 90:
    print("A Grade")
elif marks >= 75:
    print("B Grade")
elif marks >= 60:
    print("C Grade")
else:
    print("Fail")
