## 🔀 Types of Control Flow

### 🔹 if Statement

```python
age = 18

if age >= 18:
    print("You can vote")

🔹 if-else Statement
age = 16

if age >= 18:
    print("Adult")
else:
    print("Minor")

🔹 if-elif-else Ladder

marks = 85

if marks >= 90:
    print("A Grade")
elif marks >= 75:
    print("B Grade")
else:
    print("C Grade")
