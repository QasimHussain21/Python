## 🔁 Types of Loops

### 🔹 for Loop

```python
for i in range(5):
    print(i)

🔹 while Loop
i = 0

while i < 5:
    print(i)
    i += 1

⏹️ Loop Control
break → Stops loop
continue → Skips iteration

for i in range(5):
    if i == 3:
        break
    print(i)
