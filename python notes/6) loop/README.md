## 🧠 Core Concept

Loops allow your program to **repeat a block of code multiple times**.

They help automate tasks and reduce repetitive code.

---

## ⚡ What You Will Learn

* `for` loops
* `while` loops
* Loop control (`break`, `continue`)
* Iteration logic

---

## 🔁 Types of Loops

### 🔹 for Loop

```python
for i in range(5):
    print(i)
```

---

### 🔹 while Loop

```python
count = 0

while count < 5:
    print(count)
    count += 1
```

---

### 🔹 Loop Control

```python
# break example
for i in range(10):
    if i == 5:
        break
    print(i)

# continue example
for i in range(5):
    if i == 2:
        continue
    print(i)
```
